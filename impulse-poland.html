Integration

Two hosting scenarios. Full data sovereignty.

Option A · On-premise

Self-hosted on your infrastructure

The Bayes Platform is open-source and can be installed directly on the hospital's own servers. Your IT teams manage the infrastructure. No data ever leaves your perimeter.

  • Full data sovereignty — no external data flow
  • Hospital IT can call / interact with the platform via API
  • Compliant with existing hospital security policies
  • Requires IT team involvement for setup
Option B · Managed cloud

Hosted on Bayes Impact secure servers

Platform hosted on EU servers, HDS-certified (French standard; Polish equivalent to be confirmed). Integration modalities between hospital systems and ours defined contractually.

  • Fast to deploy — no IT setup required from hospital
  • Secure API / dedicated channel for HIS interaction
  • No personal data stored — session data deleted immediately after query
  • Integration agreement to be defined with hospital IT
Standard contract framework No personal data processing by default Data deleted after query No interconnection with IT system required No retraining of AI models on input data Low-risk AI classification GDPR compliant

Data Governance

A robust data governance framework, by default.

Every Impulse Healthcare contract embeds the same data governance standards — regardless of the hosting option chosen.

🔒
Personal data

No personal data processed by default

The platform is designed to operate without processing personal data. Clinical queries are anonymised before being sent to the AI layer.

🗑️
Data retention

Deleted after query

Session data is deleted immediately after each query by default. Configurable retention duration available subject to specific contract provisions.

🏥
Hosting certification

EU HDS-certified servers

Managed cloud option hosted on EU HDS-certified infrastructure (French health data standard). No interconnection with hospital IT system required.

🤖
AI model integrity

No retraining on input data

Input data is never used to retrain AI models. The AI models used remain unchanged by usage within the platform.

⚖️
AI Act classification

Low-risk AI classification

The use cases supported by Impulse are classified as low-risk under the EU AI Act. No clinical decision-making AI in scope.

🇪🇺
Regulatory compliance

GDPR compliant

Full GDPR compliance by design. Data processing agreements available. Contract framework reviewed by legal counsel.

Bayes Curriculum · Delivery

Two ways to operate the programme.

The five curriculum building blocks can be delivered in two modes — chosen based on the partner's context and capacity.

Option A · Operated by Bayes Impact

Bayes Impact designs and delivers the full curriculum directly — masterclasses, conferences, workshops, peer learning, and prototyping support.

  • End-to-end programme management by Bayes teams
  • Rapid deployment, no local capacity required
  • Delivered in English or French
Our preferred model
Option B · Train-the-trainers

Bayes Impact trains a local partner organisation to operate the five curriculum blocks autonomously — enabling long-term, locally rooted programme delivery.

  • Local presence — embedded in the hospital ecosystem
  • Polish language — full delivery in local language
  • Ecosystem connection — leverages existing clinical networks
  • Knowledge transfer — lasting capability built within the partner
  • Local insight — deep understanding of the local healthcare context
